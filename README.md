# number_game_app

My second cheap application with [react native](https://reactnative.dev/) and [expo](https://expo.io/)!

![Recordit GIF](https://github.com/trixky/number_game_app/blob/main/demo/gif.gif)

## Usage

### Requirements

- node  14.15.0
- npm   6.14.8

``` bash
npm install
npm install --global expo-cli
expo start
```
